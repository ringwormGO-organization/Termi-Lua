# Termi-Lua
Lua version of Termi

# Requirements (Minimum)

1. Lua 5.3
2. 5 KB of memory

# How to run:

## Linux

For Ubuntu/Debian
```
$ sudo apt install lua5.4
$ lua Main.lua
```

CentOS/RHEL
```
# yum install epel-release && yum install lua
# lua Main.lua
```

Fedora 22+

```
# dnf install lua
# lua Main.lua
```
